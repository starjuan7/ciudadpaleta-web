🔑 Token de GitHub

¿Cómo generar tu token?

  1. Ve a github.com/settings/tokens
  2. Clic en "Generate new token" → "Fine-grained"
  3. Nombre: ciudadpaleta-admin
  4. Expiration: 1 año (o más)
  5. Repository access: Only select repositories → tu repo
  6. Permissions → Repository → Contents: Read and write
  7. Generate → copia el token (solo se ve 1 vez)
🔒 Se guarda solo en TU navegador (encrypted localStorage). Nunca se sube a internet.
